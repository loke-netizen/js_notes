Here are **important JavaScript (JS) notes** in a **simple and easy way**:

---

## 🟡 1. Basics

* JavaScript is a **scripting language** used for web development.
* Runs in **browser + Node.js**
* Used for **frontend + backend**

---

## 🟡 2. Variables

```js
let a = 10;
const b = 20;
var c = 30;
```

* `let` → block scope (recommended)
* `const` → constant value (cannot change)
* `var` → old (avoid)

---

## 🟡 3. Data Types

* String → `"hello"`
* Number → `10, 10.5`
* Boolean → `true / false`
* Array → `[1,2,3]`
* Object → `{name:"Loke"}`
* Null / Undefined

---

## 🟡 4. Functions

```js
function add(a, b){
  return a + b;
}
```

Arrow function:

```js
const add = (a,b) => a + b;
```

---

## 🟡 5. Arrays (Important)

```js
let arr = [1,2,3];
```

Common methods:

* `push()` → add last
* `pop()` → remove last
* `shift()` → remove first
* `unshift()` → add first
* `map()` → modify each element
* `filter()` → condition check
* `reduce()` → single output

---

## 🟡 6. Objects

```js
let user = {
  name: "Loke",
  age: 20
};
```

Access:

```js
user.name
user["age"]
```

---

## 🟡 7. Loops

```js
for(let i=0;i<5;i++){}
for(let x of arr){}
for(let key in obj){}
```

---

## 🟡 8. Conditions

```js
if(age > 18){
  console.log("Adult");
} else {
  console.log("Child");
}
```

---

## 🟡 9. DOM (Important for frontend)

* `document.getElementById()`
* `querySelector()`
* `innerText / innerHTML`
* `addEventListener()`

---

## 🟡 10. Important Concepts

* Hoisting
* Scope (global, block, function)
* Closure
* Callback
* Promise
* Async/Await

---

## 🟡 11. JSON

```js
JSON.stringify(obj)
JSON.parse(json)
```
